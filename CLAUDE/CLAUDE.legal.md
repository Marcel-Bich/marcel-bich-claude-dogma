# Legal Rules

<rules>
1. **LICENSE file required** - Every public repo needs a LICENSE file in root.
2. **MIT for libraries** - Default choice for open source libraries and tools.
3. **Check third-party licenses** - Verify compatibility before using dependencies.
4. **Privacy by design** - Collect only what's needed, document in Privacy Policy.
5. **Impressum for DE** - German websites need Impressum with name, address, contact.
</rules>

For details see `GUIDES/legal.md`
